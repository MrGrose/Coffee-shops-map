# Кофейная карта

Этот мой учебный проект, который представляет собой простой скрипт на Python. Карта кофейни в пешей доступности. Принцип работы предельно прост: пользователь вводит свой текущий адрес, а приложение показывает на карте пять ближайших кофеен.

## Установка

1. Убедитесь, что у вас установлен Python 3.
2. Создайте файл `.env` в корневом каталоге проекта и добавьте следующие переменные среды:
```bash
API_KEY = ваш ключ от Yandex geocoder API
```
## Использование
1. Скачайте следующие файлы:
    - script.py - главный файл;
    - coffe.json - данные кофейн;
    - requirements.txt - библиотеки;
2. Запустите скрипт:
```bash
python ваш_скрипт.py
```
